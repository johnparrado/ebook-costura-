# 🧵 El Arte de la Costura — Ebook Profesional

Landing page / ebook interactivo de costura, 100% profesional y responsive.
Incluye consejos, técnicas y patrones de valor, con 3 llamados a la acción
discretos pero directos hacia la oferta en Hotmart.

## ✨ Características

- 📖 Contenido de valor: fundamentos, técnicas esenciales, lectura de patrones y trucos del oficio.
- 🎁 **3 contadores (countdown)** con CTA hacia la oferta de **+7.000 patrones y moldes**.
- 🛒 Botones flotantes (comprar ahora + volver arriba).
- 💎 Diseño elegante, animaciones suaves, barra de progreso, FAQ y testimonios.
- 📱 Totalmente responsive (móvil, tablet y escritorio).
- 🔗 Enlace de compra: [Hotmart](https://go.hotmart.com/X106176004X)

## 🚀 Cómo verlo

Es un único archivo `index.html` autónomo (sin dependencias). Opciones:

1. **Abrirlo directamente** en cualquier navegador.
2. **Publicarlo gratis con GitHub Pages:**
   - Ve a *Settings → Pages*.
   - En *Branch* selecciona `main` y carpeta `/root`.
   - Guarda. En unos minutos estará disponible en
     `https://johnparrado.github.io/ebook-costura-/`
